#  Vehicle Sales Dashboard – March 2026

##  Project Overview

This project is an interactive dashboard created in **Google Looker Studio** to analyze vehicle sales performance for March 2026. The dashboard provides key business insights into revenue, vehicle sales, customer preferences, salesperson performance, and financing trends.

The goal of this project was to transform raw sales data into an interactive report that helps dealership management make informed, data-driven decisions.

---

##  Business Problem

Dealerships generate large volumes of sales data each month. Without an effective reporting solution, it is difficult to monitor sales performance, identify best-selling vehicles, evaluate salesperson productivity, and understand customer purchasing trends.

This dashboard was designed to provide management with a single, interactive view of the dealership's key performance indicators (KPIs).

##  Key Insights

### Brand Performance

- Suzuki had the highest number of vehicle sales.
- Chery ranked second.
- Omoda ranked third.

### Best-Selling Models

The top-performing models included:

- Suzuki Swift 1.2 GL MT
- Suzuki Swift 1.2 GL AMT
- Omoda C5
- Omoda C9
- Tiggo 7 Pro
- Tiggo 8 Pro

### Customer Preferences

The dashboard compares customer purchasing behaviour by:

- Vehicle colour
- Transmission type (Manual vs Automatic)
- Finance provider

### Sales Performance

The report identifies top-performing branches and salespeople based on the number of vehicles sold.

### Finance Providers

The dashboard analyzes vehicle purchases financed through:

- WesBank
- MFC
- Absa Vehicle Finance
- Standard Bank Vehicle Finance
- Cash Purchases
